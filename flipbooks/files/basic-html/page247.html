( 二 ) 研討會主題
                   議題一、「日本國土交通省運輸安全管理考核 - 鐵道部分之實務作業」：
                     日本國土交通省已建立運輸安全管理制度，對日本鐵道、汽車、

                   航空、海運等各領域進行運輸安全管理評估。本次的研討內容包括
                   介紹運輸安全管理制度的設立背景、目的和主要內容等概要。關於
                   運輸安全管理評估之實務作業，於介紹評估觀點、基準、評估作業實

                   務內容的同時，也特別針對鐵道與其他領域的不同，以及考核評估面
                   之重點確認項目等進行介紹。
                   議題二、「日本鐵道邊坡維護與管理制度，暨預警及異物入侵告警系

                   統實務介紹」：
                     日本鐵道過去曾多次因大雨發生邊坡災害，為此鐵道業者持續
                   進行邊坡維護管理並採取各種防災措施，以確保列車安全。本次介紹

                   日本鐵道實施的邊坡維護管理制度和防災措施概要，包含邊坡災害
                   的實際情形和背景、鐵道邊坡的定期檢查方法、對策以及雨天的列車
                   行駛規定等。

                   議題三、「智慧鐵道系統標準架構及標準資料研討」：
                     鐵道運行的相關數據涵蓋範圍很廣，包括行駛資訊、設備狀態、
                   旅客資訊等。然而到目前為止，這些資訊都是由不同系統進行管理，

                   並未充分考慮各系統間資訊流通使用或與氣象資訊等進行統合。隨著
                   資訊處理技術的發展，以及以智慧型手機為代表的服務日新月異，
                   進而將各種數據進行整合，並建立了方便分析的數據平台。本次將
                   介紹透過此平台，如何向鐵道、周邊設施的使用民眾以及鐵道運行的
                   作業員工提供智慧化之嶄新服務。
