應用科技組 08-1                                               工作成果報告






                       壹、前言

                         可於極端嚴苛環境操作的耐高溫、耐燒蝕材料，為航太科技
                       未來發展過程中需面對的首要問題，本次研討會特邀請曾任職於 IHI
                       AEROSPACE 的超耐熱複合材料開發專家―山內宏先生，針對 SiC、

                       CMC 等複合材料的開發歷史與製程經驗進行分享，期望能夠提升本院
                       相關領域研發能量，並建立後續材料商源開發與技術合作之管道。


                       貳、研討主題
                       一、議題名稱：超耐熱複合材料關係
                       二、主講人：山內宏（第一高周波工業（株）技術顧問）
                       三、演講題目：
                            ( 一 ) 講師經歷企業能量簡介
                            ( 二 ) 複合材料發展現況與製程技術―SiC 纖維
                            ( 三 ) 複合材料發展現況與製程技術―CMC 複合材料
                            ( 四 ) 複合材料發展現況與製程技術―C/C 複合材料
                            ( 五 ) 超耐熱複合材料開發與展望


                       參、執行概況

                          本次研討會敬邀第一高周波工業（株）技術顧問―山內宏先生，
                       研討耐高溫複合材料議題，山內先生曾任職於 IHI AEROSPACE，
                       負責複合材料製程技術開發，富有關鍵製程專利開發經驗。研討會

                       於 11 月 21 日至 11 月 22 日來院內進行討論，首先由本院簡副院長
                       於 W48 館接待後，至本院飛彈所展開「超耐熱複合材料」研討會；
                       研討會由郭正山所長主持，近兩天的議程，山內先生除了開發經驗
                       分享，亦針對與會同仁提問，提供現況製程開發瓶頸突破方向，讓參加
                       會議人員獲益匪淺。
