二、 專業性觀察與建議
                   ( 一 ) 議題一「日本超高齡社會福祉住環境規劃制度及推動策略」
                      本次研討會主講人提供日本超高齡社會下高齡者住宅與政策、

                   福祉住環境協調員證照制度的設立與現況議題，包括日本福祉住環境
                   制度的推動及實施策略、日本政府扮演的角色、福祉住環境協調員角色、
                   工作、各級資格的內容概要，並對住宅設計案例提出精闢看法，同時

                   和與會專家學者、廠商、協會團體代表，進行專業意見交換，所提供之
                   資料及經驗相當值得國內參考。雖然我國法令制度與設計環境與日本
                   並不相同，但日本福祉住環境制度及福祉住環境協調員角色等議題，
                   在國內極具參考價值。

                   ( 二 ) 議題二「日本建築物設置電動車充電設備之建築防火管理與對策」
                      主講人於本次研討會提供日本建物設置電動車充電樁防火現況

                   與課題、電動車充電樁設置與防火法規、設置案例介紹等項，惟日本
                   因無類似台灣防空避難室之規定，故基於施工成本及電動車駕駛者
                   便利性等考量，較少開挖多層地下室供作停車空間之情形。雖國情
                   不同，但日本透過電動車、充電設備、建物供電系統分別嚴格控管安全
                   基準，使火災風險降至最低之作法值得效法，並和與會專家學者、

                   廠商、協會團體代表進行專業意見交換，所提供之資料及經驗相當
                   值得國內參考。

                   ( 三 ) 議題三「日本建築產業數位轉型推動策略」
                      主講人於本次研討會介紹日本中央政府推動建築產業數位轉型
                   推動之手段，並說明 BIM 是日本 DX 數位轉型研發政策的關鍵技術之
                   一；日本在「經濟財政運營與改革政策」與「科學技術創新基本

                   計畫」背景下，以 i-Construction 為基礎推動營建數位轉型計畫，並以
                   Society 5.0 為目標，實現數位治理和提升公共設施整備和維護管理                                           果 報 告 工 作 成
                   績效。演講後並與專家學者、廠商、協會團體代表進行專業意見交換，

                   所提供之資料及經驗相當值得國內參考。
